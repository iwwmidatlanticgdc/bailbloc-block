# bailbloc-block

A tiny repo to run [Bail Bloc](https://github.com/thenewinquiry/bailbloc/) on a Raspberry Pi.

Setup:

* `git clone` the repo
* `sudo apt update -y`
* `sudo apt-get install -y libcurl4-openssl-dev libjansson-dev openssl libssl`
* `cd bailbloc-block`
* `chmod +x minerd`
* `chmod +x bailblock.sh`
* `./bailbloc.sh`
